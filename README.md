tabetai_bot @mamorunomamoru1
=========

### 仕様
1.定期的につぶやく  
2.メンションを貰ったらリプライを返す。（すでにツイートした宛にはツイートしないようになっています）

### 環境
言語 : Ruby2.0  
サーバー : heroku  
データベース : MongoHQ (herokuのアドオン)
